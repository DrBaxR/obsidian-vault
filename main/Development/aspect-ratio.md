It is a [[CSS]] property that sets a preferred aspect ratio for the box - useful for whn you need something to be a square

```css
aspect-ratio: 1 / 1;
aspect-ratio: 1;

aspect-ratio: inherit;
aspect-ratio: initial;
aspect-ratio: revert;
aspect-ratio: revert-layer;
aspect-ratio: unset;
```
